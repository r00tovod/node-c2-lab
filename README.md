# Node C2 Lab 🧪

> ⚠️ **Educational Purpose Only**  
> This project is designed for learning about C2 architecture, network security, and defensive research.  
> Do not use against systems you do not own or have explicit permission to test.

## 🎯 Purpose
- Understand beacon-based communication patterns
- Practice secure coding for security tools
- Develop detection signatures for blue team training

## 🛠️ Tech Stack
- Node.js (native http/https)
- JSON-based task queue
- Bearer token authentication

## 🚀 Quick Start
1. `npm install` 
2. `node c2_server.js 8068`
3. `node operator.js`

## 🔐 Security Notes
- Use strong tokens in production
- Enable HTTPS for real deployments
- Never deploy on public networks without proper isolation
